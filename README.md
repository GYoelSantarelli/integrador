## Requisitos

Para ejecutar el proyecto es necesario tener instalado:

* Java JDK 17 o superior.
* NetBeans IDE (o cualquier IDE compatible con proyectos Java).

## Ejecución del proyecto

1. Clonar el repositorio:
En caso de no tener SSH
```bash
git clone https://github.com/GYoelSantarelli/integrador.git
```
En caso de tener SSH
```bash
git clone git@github.com:GYoelSantarelli/integrador.git
```
2. Abrir el proyecto en NetBeans (o en cualquier IDE compatible con Java).

3. Ejecutar la clase `Main.java`.

4. Utilizar el menú de consola para acceder a las distintas funcionalidades del sistema:

   * Gestión de categorías.
   * Gestión de productos.
   * Gestión de usuarios.
   * Gestión de pedidos.

## Funcionalidades principales

* Alta, baja y consulta de categorías.
* Alta, baja y consulta de productos.
* Alta, baja y consulta de usuarios.
* Creación y administración de pedidos.
* Control automático del stock.
* Cálculo automático de subtotales y totales.
